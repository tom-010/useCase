Scenario: User tries to use a username that already exists

Given a already existing user "Tom" 
And the user "Klaus" does not exist in the system
When "Klaus" tries to create a User with login "Tom"
Then a exception is triggered

Input

Requires following Input:

- username as String
- password as String
- firstname as String
- lastname as String

Output

produces

- created user as User

Steps

1. Lookup for user with the given username
  1.1 If a user was found trigger user-exist-exception
2. Create the user in the database (if error occurs: 
  trigger user-unable-to-create-exception)
3. Trigger a notification, that user was created
4. Send welcome message
